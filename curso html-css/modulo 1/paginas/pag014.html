Css Interno

Para usar css interno, vamos abrir uma tag <style> dentro da tag <head>. O recomendado é usar css interno quando so for para uma página, caso seu site tiver mais de uma página o recomendado sera usar Css externo.

Css externo é o mais recomendado, podemos criar diversas páginas com mesmos estilos.
Podemos misturar os estilos interno e os externos.
Para criar um css externo, vamos usar a a tag <link:css>, e colocar a página que o css esta localizado.
