Como colocar vídeos externos

O único problema do YT é que precisamos deixar os vídeo público, mas temos uma solução para isso, a solução é usar o Vimeo, ele permite deixar o vídeo privado, so vai poder ter acesso a ele que estiver no site. Mas é pago!
